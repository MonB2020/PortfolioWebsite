Health Planner App

May 2021 - June 2021

App Details

The Health Planner App is a simple application meant to help people easily keep track of their health.

The user fills out their information for the following health categories: Calories, Exercise, and Sleep. They also fill out their next goal for these. Additional app features include a built in BMR calculator, a food log, and a shopping list.


Features

  • Health Plan Log:
  • Fill out a health plan to log calories, exercise hours, and sleep hours
  • Save health plan log to a text file
  • Food Log:
  • Log foods eaten for each meal in a specified day
  • Save foods logged to a text file
  • Shopping List:
  • Add items and their prices to a shopping list
  • View all items currently in the shopping list
  • Delete an item from the shopping list
  • Information on exercise tips, types of activities based on intensity level, calories and sleep recommendations
  • Calculate BMR
  • Languages/Software: Java, JavaFX
  • IDE used: IntelliJ
  • UI designed with SceneBuilder

On The House

May 2022

App Details

On The House is an app that connects those in need of items to those who can give away items, to help reduce our carbon footprint one transaction at a time by reducing waste through reusing used items.


Features

  • View all users posts
  • View details of a post
  • View all your posts under the MyListings page
  • Create a post with a title, city, description, and photo
  • Edit a post's information (title, city, description, status)
  • Reserve a post item for a specific user
  • Delete a post
  • Message another post owner
  • View your profile information
  • Edit your profile information
  • View another user's profile
  • Rate another user

Features I Contributed To

  • Signup feature
  • Security checks for login, signup, creating and editing a post
  • Reserve a post item for a specific user
  • Popup confirmation messages for deleting a post and updating profile information
  • Private messaging using socket.io
  • Editing profile information
  • User dashboard with features to add, edit, and delete a user (for admin users only)
  • Languages/Software: HTML, CSS, JavaScript, Node.js, Express, MySQL
  • IDE used: Visual Studio Code
  • Hosting: Heroku
  • Created in a group of 4
